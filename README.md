# Deadlock Prevention & Recovery Toolkit

## 📌 Overview
This project is a Streamlit-based interactive toolkit designed to demonstrate:
- Deadlock Detection
- Deadlock Prevention
- Deadlock Recovery

## ⚙️ Technologies Used
- Python
- Streamlit
- NumPy
- Pandas
- Plotly

## 🚀 How to Run
```bash
streamlit run app.py
